Quiz Master 🧠
A simple and interactive quiz application built using HTML, CSS, and JavaScript. Users can register, log in, answer multiple-choice questions, track their scores, and view a leaderboard.
🚀 Features
🔐 User Registration & Login
📝 Multiple Choice Quiz Questions
⏱️ Countdown Timer for Each Question
📊 Score Tracking
🏆 Leaderboard System
💾 Local Storage Support
🎨 Responsive and Modern UI

🎮 How to Play
Open index.html.
Click Start Quiz.
Register a new account or log in.
Answer the quiz questions before the timer runs out.
View your final score.
Check your ranking on the leaderboard.


⏱️ Quiz Rules
Each question has a timer.
Select the correct answer before time expires.
Correct answers increase your score.
Final scores are saved to the leaderboard.
